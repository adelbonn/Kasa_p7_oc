# Kasa

## Description

Application de location immobilière développée avec React et Vite.
Le projet vise à moderniser le site de Kasa, une entreprise de location d'appartements entre particuliers, en utilisant un stack Javascript
(projet 7 du parcours Intégrateur Web d'Openclassrooms)


## Technologies utilisées

- ** Frontend : ** React, Vite, React Router
- **Backend : ** Données simulées à partir d'un fichier json
- **Outils : Git, Github

## Fonctionnalités

- Affichage des annonces de location avec des données dynamiques
- Navigation entre les pages avec React Router
- Composants réutilisables et responsives basés sur la maquette figma
- Gestion des états et des animations CSS pour une meilleure expérience utilisateur
